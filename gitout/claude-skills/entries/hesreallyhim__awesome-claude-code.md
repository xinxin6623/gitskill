## 💡 解决什么问题

Claude Code 生态爆炸式增长，你不知道：

- 别人都在用什么 skill / hook
- 哪些已经验证好用、哪些是水货

awesome 列表的价值就是**有人替你筛过一遍**——这个项目强调"代码质量、安全性、原创性"三条标准，不是无脑收录。
